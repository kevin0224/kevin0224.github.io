html入门标签

October 18, 2019

html入门笔记1

  1. html是由 李爵士 发明的
  2. html由 <!DOCTYPE空格html> 起手 表示 文档类型
  3. 常用的章节标签
    • 标题 h1-h6
    • 章节 section
    • 段落 p
    • 头部内容 header
    • 脚部内容 footer
    • 主要内容 main
    • 旁支内容 aside
    • 划分 div
  4. 全局属性
    • class 用.选择器
    • contenteditable 使任何一个元素可以被编辑
    • hidden 隐藏消失
    • id 用#选择器 全页面唯一 尽量不用
    • style 同时存在以js为最终效果
    • tabindex 控制tab访问顺序 =0最后访问 =-1不访问
    • title 显示完整的内容
  5. 常用的内容标签
    • ol li 有序标签
    • ul li 无序标签
    • dl dt dd dt表示被描述的词 dd表示详细描述的内容
    • pre 连续空格tab回车会被缩进成一个空格 pre可以保留多个空格
    • code 中的代码是等宽的
    • hr 分割线
    • br 换行
    • a标签 超链接
    • em 强调 语气
    • strong 强调 本质
    • q quote 引用 内联
    • blockquote 引用 一块的引用
